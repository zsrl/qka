# 报告

回测跑完，看结果。QKA 提供两种方式：终端概览和 HTML 报告。

---

## summary — 终端概览

```python
bt.summary()
```

输出：

```
QKA 回测报告 — MyStrategy
────────────────────────────────────────
初始资金:        ¥100,000.00
最终资产:        ¥178,233.45
总收益率:        +78.23%
年化收益率:      +11.34%
最大回撤:        -32.15%
夏普比率:        0.68
交易次数:        47
胜率:            61.70%
────────────────────────────────────────
```

### 指标含义

| 指标 | 什么意思 |
|------|---------|
| 总收益率 | 从第一天到最后一天，总资产涨了百分之多少 |
| 年化收益率 | 折算成每年收益率，方便对比不同时间长度的策略 |
| 最大回撤 | 历史上从最高点跌到最低点的最大幅度。越小越好 |
| 夏普比率 | 每承担一份风险能换多少收益。> 1 算不错，> 2 很好 |
| 胜率 | 赚钱的交易占总交易的比例。高胜率 ≠ 好策略 |
| 交易次数 | 总交易笔数。次数太少的话胜率和收益率参考意义不大 |

加了 benchmark 后还会多出：

| 指标 | 什么意思 |
|------|---------|
| 超额收益 | 策略比沪深 300 多赚了多少 |
| 超额夏普 | 超额收益的夏普比率 |

## report — HTML 报告

```python
bt.report(title='我的策略', output_path='./my_report.html')
```

浏览器自动打开。不传 `output_path` 的话，默认存到 `./reports/`。

### 报告里有什么

- **绩效指标卡片** — summary 里的所有数字，卡片展示
- **净值曲线** — 资产变化曲线，带基准对比（如果有 benchmark）
- **月度收益热力图** — 每月收益百分比，绿色赚钱红色亏钱，一眼看出哪些月份表现好
- **交易明细列表** — 每笔交易的买卖时间、价格、股数、盈亏

### 手机也能看

报告是自包含的 HTML 文件，纯前端渲染（Plotly），不需要开服务器。双击就能打开，发到微信、邮件里在手机上打开也一样能看。

---

## 小提示

- `bt.report()` 返回 `Path` 对象，可以用 `str(report_path)` 拿到文件路径
- 多次调用 `bt.run()` 会覆盖之前的报告
- 交易次数太少时，胜率 100% 也可能是蒙的，结合交易次数一起看
